# Politique de confidentialité – AppReel

**Dernière mise à jour : mars 2026**

Cette politique de confidentialité décrit la manière dont l’application AppReel (« l’application », « nous ») collecte, utilise et protège les informations dans le cadre de l’utilisation de l’application mobile.

## 1. Responsable du traitement

Les données sont traitées dans le cadre de l’utilisation de l’application AppReel. Pour toute question relative à cette politique ou à vos données, vous pouvez nous contacter via les coordonnées indiquées dans la fiche de l’application sur le store.

## 2. Données collectées et finalités

### 2.1 Données liées à l’utilisation de l’application

- **Données locales** : l’application peut enregistrer localement sur votre appareil des informations liées à votre utilisation (par exemple historique de consultation, préférences) afin d’améliorer votre expérience. Ces données restent sur votre appareil sauf si vous réinitialisez l’application ou les données de l’appareil.

- **Données techniques** : des informations techniques (type d’appareil, système d’exploitation, langue) peuvent être utilisées pour assurer le bon fonctionnement et la compatibilité de l’application.

### 2.2 Publicité

L’application affiche des publicités fournies par des partenaires tiers. Selon la version ou la configuration de l’application, un ou plusieurs des services suivants peuvent être utilisés. Ces partenaires peuvent collecter et traiter des données pour diffuser des annonces, mesurer leur efficacité et personnaliser le contenu publicitaire.

#### AppLovin (MAX)

L’application peut utiliser le SDK AppLovin MAX pour afficher des publicités (notamment publicités récompensées). AppLovin et ses partenaires peuvent collecter ou recevoir des données telles que :

- Identifiant publicitaire de l’appareil (GAID sur Android, IDFA sur iOS)
- Adresse IP
- Données d’utilisation de l’application
- Données techniques (modèle d’appareil, système d’exploitation, langue)

**Finalités** : diffusion et personnalisation des annonces, mesure des performances publicitaires, prévention de la fraude.

- Politique de confidentialité d’AppLovin : [https://www.applovin.com/privacy/](https://www.applovin.com/privacy/)

#### Google (AdMob)

L’application peut utiliser Google Mobile Ads (AdMob) pour afficher des publicités. Google peut collecter ou recevoir des données telles que :

- Identifiant publicitaire de l’appareil (GAID sur Android, IDFA sur iOS)
- Adresse IP
- Données d’utilisation de l’application
- Données techniques

**Finalités** : diffusion et personnalisation des annonces, mesure des performances, analytics.

- Politique de confidentialité de Google : [https://policies.google.com/privacy](https://policies.google.com/privacy)  
- Informations AdMob et confidentialité : [https://support.google.com/admob/answer/6128543](https://support.google.com/admob/answer/6128543)

## 3. Base légale et consentement

- Pour les publicités personnalisées, le traitement peut reposer sur votre consentement (par exemple via les paramètres de suivi sur votre appareil ou les choix proposés dans l’application).
- Vous pouvez à tout moment retirer votre consentement ou limiter le ciblage publicitaire via les paramètres de votre appareil (ex. « Limiter le suivi publicitaire » sur iOS, paramètres publicité sur Android).

## 4. Conservation des données

- Les données stockées localement sur votre appareil sont conservées tant que l’application est installée (ou jusqu’à réinitialisation).
- Les données traitées par les partenaires publicitaires sont soumises à leurs propres politiques de conservation (voir les liens ci-dessus).

## 5. Vos droits

Selon la réglementation applicable (RGPD, CCPA, etc.), vous pouvez disposer de droits tels que : accès à vos données, rectification, suppression, limitation du traitement, opposition, portabilité. Pour les données traitées par des tiers (AppLovin, Google), vous pouvez exercer vos droits auprès de ces partenaires ou via les paramètres de votre appareil.

## 6. Sécurité

Nous mettons en œuvre des mesures appropriées pour protéger les données traitées dans le cadre de l’application. Les partenaires tiers sont tenus de respecter leurs propres obligations en matière de sécurité et de confidentialité.

## 7. Modifications

Cette politique peut être mise à jour. La date de dernière mise à jour est indiquée en tête de document. Une version à jour sera accessible dans l’application et, le cas échéant, via l’URL fournie sur la fiche de l’application sur les stores.

## 8. Contact

Pour toute question relative à cette politique de confidentialité ou à vos données personnelles, veuillez utiliser les coordonnées de contact indiquées sur la fiche de l’application (App Store / Play Store) ou dans l’application.
